Profound gives you programmatic access to AI visibility data — how often your brand appears in ChatGPT, Perplexity, Google AI Overviews, and more. This quickstart walks you through installing an SDK, authenticating, and making your first API call in under five minutes.

## Prerequisites

Before you begin, you need:

- A Profound account ([sign up at tryprofound.com](https://www.tryprofound.com))
- An API key from your [dashboard](https://app.tryprofound.com) (Settings → API Keys)
- Node.js 20+ (for TypeScript) or Python 3.9+ (for Python)

<Steps>
  <Step title="Get your API key">
    Log in to the [Profound dashboard](https://app.tryprofound.com), navigate to **Settings → API Keys**, and create a new key. Copy it and store it in your environment:

    ```bash
    export PROFOUND_API_KEY="your_api_key_here"
    ```

    <Note>
      Never hardcode your API key in source code. See [Authentication](/authentication) for best practices on credential management.
    </Note>
  </Step>

  <Step title="Install the SDK">
    Profound provides official SDKs for TypeScript/JavaScript and Python. Choose the package manager that matches your project.

    <CodeGroup>

      ```bash npm
      npm install @profoundai/client
      ```

      ```bash yarn
      yarn add @profoundai/client
      ```

      ```bash pnpm
      pnpm add @profoundai/client
      ```

      ```bash pip
      pip install profound
      ```

    </CodeGroup>

    <Tip>
      If you prefer to call the API directly without an SDK, every example in this guide also works with plain HTTP. Check the [API Reference](/api-reference/overview) for full endpoint documentation.
    </Tip>
  </Step>

  <Step title="Make your first API call">
    Fetch the list of tracked categories for your organization. Categories represent the product lines, topics, or competitive segments you're monitoring across AI assistants.

    <CodeGroup>

      ```ts TypeScript
      import Profound from '@profoundai/client';

      const client = new Profound({
        apiKey: process.env['PROFOUND_API_KEY'],
      });

      // List the categories tracked by your organization
      const categories = await client.organizations.categories.list();
      console.log(categories);
      ```

      ```python Python
      import os
      from profound import Profound

      client = Profound(
          api_key=os.environ.get("PROFOUND_API_KEY"),
      )

      # List the categories tracked by your organization
      categories = client.organizations.categories.list()
      print(categories)
      ```

    </CodeGroup>

    If you see a `401 AuthenticationError`, double-check that `PROFOUND_API_KEY` is set correctly in your shell environment.
  </Step>

  <Step title="Pull a visibility report">
    Now fetch a visibility report to see how often your brand is cited — and with what sentiment — across AI assistants for a given date range. Pass `start_date` and `end_date` in `YYYY-MM-DD` format.

    <CodeGroup>

      ```ts TypeScript
      import Profound from '@profoundai/client';

      const client = new Profound({
        apiKey: process.env['PROFOUND_API_KEY'],
      });

      // Pull a visibility report
      const visibility = await client.reports.visibility({
        start_date: '2024-01-01',
        end_date: '2024-01-31',
      });
      console.log(visibility);
      ```

      ```python Python
      import os
      from profound import Profound

      client = Profound(
          api_key=os.environ.get("PROFOUND_API_KEY"),
      )

      # Pull a visibility report
      visibility = client.reports.visibility(
          start_date="2024-01-01",
          end_date="2024-01-31",
      )
      print(visibility)
      ```

    </CodeGroup>

    The response includes citation counts, share-of-voice percentages, and sentiment scores broken down by AI assistant and category. Use these metrics to benchmark your brand's AI presence over time.
  </Step>
</Steps>
