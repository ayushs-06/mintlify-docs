---

The Bots Report gives you a per-model breakdown of AI activity tracked by Profound — showing which bots are querying topics in your space, how many answers each model produces, and how your brand's visibility compares across the AI landscape. Use it to understand which models deserve the most optimization attention.

Profound provides two versions of this report. **v1** (`getBotsReport`) returns the standard bots dataset. **v2** (`getBotsReportV2`) returns an enhanced schema with additional engagement and coverage fields.

## Request Parameters

<ParamField query="start_date" type="string" required>
  The start of the reporting window in `YYYY-MM-DD` format.
</ParamField>

<ParamField query="end_date" type="string" required>
  The end of the reporting window in `YYYY-MM-DD` format.
</ParamField>

<ParamField query="bot_name" type="string">
  Filter results to a single AI bot by name. Omit to aggregate across all tracked bots.
</ParamField>

<ParamField query="model_id" type="string">
  Filter by a specific model version.
</ParamField>

<ParamField query="region_id" type="string">
  Restrict results to a specific geographic region.
</ParamField>

<ParamField query="category_id" type="string">
  Scope the report to a single prompt category.
</ParamField>

<ParamField query="persona_id" type="string">
  Filter by user persona.
</ParamField>

<ParamField query="prompt_type" type="string">
  Limit to a specific prompt type.
</ParamField>

<ParamField query="tag_id" type="string">
  Filter by tag.
</ParamField>

<ParamField query="topic_id" type="string">
  Narrow to a specific topic.
</ParamField>

<ParamField query="asset_id" type="string">
  Filter to a specific tracked content asset.
</ParamField>

<Note>
  Use v2 (`getBotsReportV2`) for richer data including additional coverage and engagement breakdowns. v1 remains available for backwards compatibility.
</Note>

## Code Examples

### v1 Report

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.profound.ai/v1/reports/bots?start_date=2025-01-01&end_date=2025-01-31" \
  --header "Authorization: Bearer $PROFOUND_API_KEY"
```

```typescript TypeScript
import { ProfoundClient } from "@profoundai/client";

const client = new ProfoundClient({ apiKey: process.env.PROFOUND_API_KEY });

const report = await client.reports.getBotsReport({
  start_date: "2025-01-01",
  end_date: "2025-01-31",
});

console.log(report);
```

```python Python
import os
from profound import ProfoundClient

client = ProfoundClient(api_key=os.environ["PROFOUND_API_KEY"])

report = client.reports.get_bots_report(
    start_date="2025-01-01",
    end_date="2025-01-31",
)

print(report)
```

</CodeGroup>

## Response Fields

<ResponseField name="data" type="array">
  List of per-bot activity records for the reporting window.

  <Expandable title="data[]">
    <ResponseField name="bot_name" type="string">
      The name of the AI bot or assistant (e.g., `"chatgpt"`, `"gemini"`, `"claude"`).
    </ResponseField>

    <ResponseField name="model_id" type="string">
      The specific model version identifier, if available.
    </ResponseField>

    <ResponseField name="answer_count" type="integer">
      Total number of answers generated by this bot during the reporting window.
    </ResponseField>

    <ResponseField name="visibility_score" type="number">
      Your brand's visibility score within this bot's answers.
    </ResponseField>

    <ResponseField name="mention_count" type="integer">
      Number of answers in which your brand was mentioned.
    </ResponseField>

    <ResponseField name="citation_count" type="integer">
      Number of times this bot cited one of your tracked URLs.
    </ResponseField>

    <ResponseField name="coverage_rate" type="number">
      Proportion of prompts in scope where this bot returned an answer containing your brand. Expressed as a decimal (e.g., `0.42` for 42%). Available in v2 only.
    </ResponseField>

    <ResponseField name="region_id" type="string">
      The geographic region for this record, if filtered.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Metadata describing the report parameters and totals.

  <Expandable title="meta">
    <ResponseField name="start_date" type="string">
      The start date used for this report.
    </ResponseField>

    <ResponseField name="end_date" type="string">
      The end date used for this report.
    </ResponseField>

    <ResponseField name="total_bots" type="integer">
      Total number of distinct bots represented in the results.
    </ResponseField>

    <ResponseField name="version" type="string">
      Report schema version — `"v1"` or `"v2"`.
    </ResponseField>
  </Expandable>
</ResponseField>
